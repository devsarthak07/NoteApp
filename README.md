# 📝 NoteApp

A simple and minimal Notes Taking Android App built with React Native and Expo.

## ✨ Features

- 📝 Create, edit and delete notes
- ✅ Checklist / to-do items inside notes
- 🎤 Voice recording attached to notes
- 🖼️ Image attachments from gallery
- 📄 Export notes as PDF
- 🔍 Search notes
- 🎨 Colorful minimal Material UI
- ✨ Smooth screen animations

## 🛠️ Tech Stack

- React Native (Expo SDK 51)
- AsyncStorage for local storage
- React Navigation for screen routing
- expo-av for voice recording
- expo-image-picker for image attachments
- expo-print for PDF export

## 🚀 Getting Started

### Prerequisites
- Node.js v20
- Expo Go app or Android Emulator

### Installation
```bash
git clone https://github.com/devsarthak07/NoteApp.git
cd NoteApp
npm install
npx expo start
```
